Flex grid

PrimoCSS includes a responsive mobile first fluid flex box grid that scales up to 12 columns or be changed for more or less dependant on your framework needs.

The predefined classes give any project the flexibility to create any layout required as well as screen specific classes.

<div class="grid-row">
    <div class="grid-col-6">
        ...
    </div>
    <div class="grid-col-6">
        ...
    </div>
</div>
