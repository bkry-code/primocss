Primo CSS — Grid

Flex grid

PrimoCSS includes a responsive mobile fluid flex-box grid that scales up to 12 columns or can be changed via the scss variable $grid-max-columns.

The predefined classes give any project the flexibility to create any layout required as well as screen specific classes. Settings .grid-md-col-6 will set the width for min-width of medium screens.

<div class="grid-row">
    <div class="grid-md-col-6">
        ...
    </div>
    <div class="grid-md-col-6">
        ...
    </div>
</div>

Offset classes

Creates an offset left margin to mimick column space between itself and another grid element.

<div class="grid-row">
    <div class="grid-col-5 offset-col-7">
    ...
    </div>
</div>
