Grid

Container

Containers are the basic level of layout elements for PrimoCSS. They're used to hold the grid layout and can be extended for other requirements.

<div class="container">...</div>

Fluid container

Using the modifier class .container--fluid for setting a full width container.

<div class="container container--fluid">...</div>

Strip container

The class .container--strip sets padding on the top and bottom plus a border to split content. 

<div class="container container--strip">...</div>

Cover container

Cover containers can take the full height, half or third dependant on screen / browser size.

<div class="container container--cover">...</div>
<div class="container container--cover-half">...</div>
<div class="container container--cover-third">...</div>

Inconjunction with the .container--middle will vertically centre any content. An example can be found on Landing.html.

<div class="container container--cover container--middle">...</div>

Color containers

Color modifiers will set a whole container background. Colors that can be used are;

Red, green, blue, yellow, orange, navy, grey, black and white.

Container with an orange background.

<div class="container container--orange">...</div>

Flex grid

PrimoCSS includes a responsive mobile fluid flex-box grid that scales up to 12 columns or can be changed via the scss variable $grid-max-columns.

The predefined classes give any project the flexibility to create any layout required as well as screen specific classes. Settings .grid-md-col-6 will set the width for min-width of medium screens.

<div class="grid-row">
    <div class="grid-md-col-6">
        ...
    </div>
    <div class="grid-md-col-6">
        ...
    </div>
</div>

Offset classes

Creates an offset left margin to mimick column space between itself and another grid element.

<div class="grid-row">
    <div class="grid-col-5 offset-col-7">
    ...
    </div>
</div>

Nesting grids

Grid has the ability to nest grids with inside any column. Repeat a .grid-row and a new 12 column grid can be made.

<div class="grid-row">
    <div class="grid-col-12">
        <div class="grid-row">
            <div class="grid-col-6">
                ...
            </div>
            <div class="grid-col-6">
                ...
            </div>
        </div>
    </div>
</div>

No gutters

Applying the class .grid-row--no-gutter will remove margins from the row and all columns.

<div class="grid-row grid-row--no-gutter">
    <div class="grid-col-12">
        ...
    </div>
</div>
