A simple and mobile friendly HTML and CSS framework

Create something awesome.

Or install the package npm install primocss.

Built for ease

PrimoCSS is a framework which provides the prime starting blocks for any front-end requirements. Without supplying out of the box solutions or mantras to follow, it creates a solid architecture footing to complete any project.

Install as a dependancy

Via npm or yarn you can include the PrimoCSS source files as a dependancy in your web project. The package only includes the sorce SCSS.

View install methods

Download compiled

If you require a quick start with no compiling, download and include it on your project.

<link href="https://fonts.googleapis.com/css?family=Open+Sans:300,300i,400,400i,600,600i,700,700i,800,800i" rel="stylesheet">

Great starting point

All the styles and tools have been created as the perfect starting point for your project. This isn't a full framework, trying to impose design styles and mantras.
